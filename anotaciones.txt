1. Instalacion del backend
se utilizan los siguientes comandos
-npm install express-generator -g
-express --view=pug "se coloca en nombre del backend"
Luego ingreso a la carpeta del backend y se ejecuta el comando
-npm install // Actualiza y agrega las dependencias faltantes
-npm install --save sequelize
